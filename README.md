🌟 What is UPSCALE IT?
UPSCALE IT is a premium, gamer-first Mod Manager designed to revolutionize your PC gaming performance. It seamlessly integrates the most powerful upscaling and frame generation mods into your games with a single click. Whether you want to force Frame Generation on older GPUs, update your native DLSS/FSR files, or compress massive game folders without losing performance, UPSCALE IT does it all through a stunning, highly optimized dark-mode interface.

🚀 Key Features
Unlock DLSS Frame Generation (FG) on any GPU! DLSS Enabler translates DLSS3 Frame Generation into AMD's FSR3 Frame Generation. Seamlessly integrate FSR3 FG into any DX12 game that natively supports DLSS3 FG.

Includes intelligent .ini configuration management.
Works on RTX 20/30 series and non-RTX cards!
Trick your games into thinking you have an RTX GPU! OptiScaler intercepts native DLSS inputs and redirects them to XeSS or FSR2/3. This allows users with GTX, AMD, or Intel GPUs to utilize DLSS upscaling logic for massive performance boosts.

Automatic API fetching for the latest versions.
Smart injection techniques (dxgi.dll, wininet.dll, etc.).
Stop playing with outdated upscaling files! Many games ship with old, blurry versions of DLSS.

Automatically fetches and replaces nvngx_dlss.dll and nvngx_dlssg.dll with the latest official releases.
Improves image quality and drastically reduces ghosting.
Running out of storage? Compress massive game directories transparently at the disk level using Windows 10/11 native WOF (Windows Overlay Filter) technology.

Save gigabytes of space with virtually zero performance impact.
Smart Exclusion Algorithm: Automatically excludes sensitive mod files (like OptiScaler/DLSS Enabler .dll and .ini files) to ensure flawless compatibility with anti-cheats and mod loaders.
Auto-Discovery: Automatically scans Steam, Epic Games, Xbox, and EA directories to find your games.
Instant Caching: Blazing fast load times thanks to an optimized JSON caching system.
Cover Art Fetching: Automatically retrieves high-quality cover art for your installed games.
Drag & Drop Reordering: Organize your game library visually in a split second.
